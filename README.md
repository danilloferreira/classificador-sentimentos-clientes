# Classificador Inteligente de Sentimentos em Comentários de Clientes

Projeto final da disciplina de Inteligência Artificial da Universidade Federal do Oeste do Pará.

## Descrição

Este projeto tem como objetivo desenvolver um classificador inteligente de sentimentos em comentários de clientes. O sistema classifica comentários em três categorias:

- Positivo
- Negativo
- Neutro

A solução utiliza técnicas de aprendizado de máquina supervisionado, pré-processamento textual, vetorização TF-IDF e modelo de Regressão Logística.

## Tecnologias utilizadas

- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
- Google Colab

## Base de dados

A base de dados utilizada foi criada de forma simulada, contendo comentários de clientes classificados manualmente como positivos, negativos ou neutros.
